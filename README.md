# WSJ English Study

월스트리트 저널(WSJ)로 영어를 공부하는 단일 HTML 웹앱입니다.
`index.html` 하나만 열면 바로 사용할 수 있습니다.

## 주요 기능
- 📰 **리딩 라이브러리** — 섹션 필터·전체 검색, 읽기 시간·단어 수 표시
- 🔤 **단어 학습** — 단어를 탭하면 발음·영영 정의·한국어 번역
- ✨ **핵심 어휘 강조**, 🔊 **전체 낭독**, 🇰🇷 **문장별 번역**
- 📒 **단어장** — 플래시카드·퀴즈·내보내기
- ➕ **새 신문 넣기** — PDF 드래그&드롭 또는 텍스트 붙여넣기
- ☁️ **기기 간 동기화** — 내 GitHub 저장소에 데이터를 저장해 모든 컴퓨터에서 연동, 변경마다 커밋 기록이 남음
- 🌙 라이트/다크 테마, 📱 모바일 대응

## 동기화 설정 (한 번만)
1. 앱 상단 **☁️ 동기화** 열기
2. [Fine-grained 토큰 만들기](https://github.com/settings/personal-access-tokens/new)
   - Repository access → 이 저장소 선택
   - Permissions → **Contents: Read and write**
3. 앱에 `저장소(owner/repo)` 와 `토큰` 입력 → **연결하고 동기화**
4. 다른 컴퓨터에서도 같은 저장소·토큰을 넣으면 단어장·기록·불러온 신문이 그대로 연동됩니다.

> 토큰은 각 브라우저에만 저장되며, 저장소나 HTML 파일에는 절대 기록되지 않습니다.
> 학습 데이터는 `wsj-study/` 폴더에 JSON으로 저장됩니다.

## GitHub Pages로 열기 (선택)
저장소 **Settings → Pages → Branch: main** 으로 설정하면
`https://<사용자명>.github.io/wsj/` 에서 바로 접속할 수 있습니다.
