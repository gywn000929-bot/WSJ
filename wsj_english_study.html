또는 텍스트 직접 붙여넣기

☁️ 기기 간 동기화

Firebase에 데이터를 저장하면, 어느 기기든 앱을 열기만 해도 토큰·링크 없이 자동으로 저장·연동됩니다.

🔌 아직 연결되지 않았습니다.
⚙️ Firebase 만드는 법 (처음 한 번, 약 5분 · 무료)
  1. Firebase 콘솔 접속 → 프로젝트 만들기 (이름 아무거나, 애널리틱스는 꺼도 됨)
  2. 왼쪽 메뉴 빌드 → Realtime Database → 데이터베이스 만들기 → 위치 선택 → 테스트 모드로 시작
  3. 만들어진 화면 위쪽의 주소(https://…firebaseio.com)를 복사해 위 “Database URL”에 붙여넣기
  4. ☁️ 자동 백업 켜기 누르면 끝. 다른 기기에서는 같은 URL만 넣으면 자동 연동됩니다.

더 편하게: 이 URL을 저에게 알려주시면 앱에 넣어 드려요. 그러면 어느 기기든 그냥 열기만 해도 자동 연동됩니다(입력조차 불필요). Firebase URL은 공개돼도 안전한 값입니다. 오래 안전하게 쓰려면 나중에 규칙(Rules)만 잠가두면 됩니다.

🛠 고급: GitHub 저장소로 백업하기

GitHub 방식은 커밋 기록이 남지만, 기기마다 토큰(또는 링크)이 한 번 필요합니다. 링크에는 토큰이 들어 있으니 내 기기에만 보내세요.
