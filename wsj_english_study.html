☁️ 기기 간 동기화

내 GitHub 저장소에 학습 데이터를 저장하면 모든 컴퓨터에서 연동되고, 변경마다 커밋 기록이 남습니다.

🔌 아직 연결되지 않았습니다.
🔑 토큰 만드는 법 (처음 한 번, 약 2분)
  1. GitHub 로그인 → Fine-grained token 만들기 열기
  2. Token name 아무거나 · Expiration 원하는 기간
  3. Repository access → “Only select repositories” → 이 학습용 저장소 선택
  4. Permissions → Repository permissions → Contents 를 Read and write 로
  5. Generate token → 나온 토큰(github_pat_…)을 위 “액세스 토큰”에 붙여넣기

저장소는 비공개(private)를 권장합니다. 토큰은 저장소나 HTML에 절대 기록되지 않고, 이 브라우저 안에만 보관됩니다. 다른 사람 컴퓨터에서는 각자 토큰을 넣어야 합니다.
